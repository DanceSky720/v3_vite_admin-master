# PreviewOrEdit 问卷预览编辑面板

## 1. 介绍 🤔

为问卷提供编辑或预览的组件

## 2. 组件说明 🥳

### 2.1 Props

v-model 代表 modelValue / v-model

| 参数           | 说明     | 类型     | 默认值 | 引入版本 |
| -------------- | -------- | -------- | ------ | -------- |
| v-model `必需` | 问卷实体 | object |        | 0.1.0    |
| preview      | 预览模式 | boolean  | false  | 0.1.0    |
| questionnaireType | 问卷类型数组 | array  | []        | 0.1.0    |

### 2.2 Events

| 事件 | 说明             | 回调参数 | 引入版本 |
| ---- | ---------------- | -------- | -------- |
| save | 保存按钮点击触发 | 无       | 0.1.0    |
