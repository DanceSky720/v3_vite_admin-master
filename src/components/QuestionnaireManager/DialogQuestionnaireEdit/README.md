## 2. 组件说明 🥳

### 2.1 Props

v-model 代表 modelValue / v-model

| 参数                | 说明     | 类型    | 默认值 | 引入版本 |
| ------------------- | -------- | ------- | ------ | -------- |
| v-model `必要` | 问卷     | object  |        | 0.1.0    |
